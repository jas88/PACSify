# PACSify

[![Build and test](https://github.com/jas88/PACSify/actions/workflows/dotnet.yml/badge.svg)](https://github.com/jas88/PACSify/actions)
[![codecov](https://codecov.io/gh/jas88/PACSify/graph/badge.svg)](https://codecov.io/gh/jas88/PACSify)

A PACS integration tool for exchanging DICOM data and processing it through local algorithms automatically.

On Windows NT and Linux (systemd) platforms this now runs as a service daemon.
